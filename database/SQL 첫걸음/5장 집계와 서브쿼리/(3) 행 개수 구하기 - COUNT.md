## COUNT로 행 개수 구하기

SQL은 집합을 다루는 집계 함수를 제공합니다. 집계란, 집합으로부터 **하나의 값**을 계산하는 것을 의미합니다. 일반적인 함수는 인수로 하나의 값을 지정하는 데 비해 집계 함수는 인수로 집합을 지정합니다. 

### 집계함수와 NULL

집계 함수는 집합 안에 NULL 값이 있을 경우 이를 **제외**하고 처리합니다.
